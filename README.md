# 🏦 End-to-End Banking Data Engineering Project

## 📌 Project Overview

This project demonstrates how to build a production-ready Data Engineering pipeline using Azure, Databricks, PySpark, Delta Lake, SQL Server, and Power BI.

The project simulates a real banking environment where data is collected from multiple sources, transformed through the Medallion Architecture (Bronze, Silver, Gold), and delivered to analytics dashboards.

---

## 🎯 Project Goals

- Build a production-style Data Engineering pipeline.
- Implement Medallion Architecture.
- Process structured and semi-structured data.
- Apply Delta Lake best practices.
- Create analytics-ready datasets.
- Build Power BI dashboards.
